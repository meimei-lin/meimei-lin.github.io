07 最佳資費選擇

輸入每月網內、網外、市話、通話時間(秒)及網內、網外簡訊則數、網路流量，求出最佳資費。

優惠內容月租費可抵等額通信費 : 若總通信費(包含通話時間、簡訊數量、網路加購)高於該資費類型的費用，則以原費用進行收費，若低於該資費類型的費用，則以該資費類型的費用進行收費。

例如: 選擇183型，總通信費為200元，則應繳金額為200元，若總通信費為150元，則應繳金額為183元。

每種方案都要計算過一次，再去比較最小費用符合哪一種資費類型。

資費類型的費率表：
資費類型 183型 383型 983型 1283型
月租費 183元 383元 983元 1283元

簡訊費率：
(元/則) 網內 1.128 1.128 1.128 1.128

流量 1G 3G 5G 吃到飽

                  #計算183類型
                  def Adopt183(speakInside,speakOutside,home,messageInside,messageOutside,network):

                    #檢查網路流量是否有超過1G
                    if network < 1:
                        network = 0
                    else: #有超過，計算超出多少G
                        network = network - 1
                
                    total = speakInside*0.08 \
                            + speakOutside*0.139 \
                            + home*0.135 \
                            + messageInside*1.128 \
                            + messageOutside*1.483 \
                            + network*250

                    if(total<=183):
                        return 183
                    else:
                        return int(total)
