23 撲克牌十點半



電腦與玩家進行撲克牌比大小，兩方輪流發撲克牌，結束時加總點數最接近10.5者贏。

點數：牌面A~10 點數為 1~10,J, K, Q 為 0.5。

規則：
1. 第一回合雙方各發一張牌
2. 除了第一回合外，玩家每回合開始可以決定是否要牌
3. 任一方點數超過10.5，即分數歸零，另一方獲勝
4. 任一回合並未要牌的一方，之後回合失去要牌權利
5. 電腦判斷要牌標準：
 1. 總點數比玩家小 或 2. 總點數8點以下(含)
6. 結束時雙方點數相同則平手

輸入說明 輸出說明
1. 輸入一個字元，表示玩家獲得的第一張牌。
2. 輸入一個字元，表示電腦獲得的第一張牌。
3. 輸入一個字元，表示玩家是否繼續要牌。
4. 輸入一個字元，表示如果玩家繼續要牌，玩家在這回合所獲得的牌。
5. 輸入一個字元，表示如果電腦繼續要牌，電腦在這回合所獲得的牌。
每回合重複第三行到第五行
注意: 第三行到第五行是可選的，依據電腦或玩家是否要牌。 		1. 輸出勝負結果字串。
可能輸出：
 1.computer win
 2.player win
 3.it's a tie

  若<條件式>成立，則進入迴圈內執行<敘述區段A>一次；
  否則離開迴圈，執行敘述區段B

            敘述區段B

副程式-deal():處理發牌流程 

        def deal(myPoint, bPoint, judge):
            over = False
            #judge(參數1,參數2)是指可以接受兩個參數，並傳回True或False的函數
            #也就是在judge這個參數位置可以放playerJudge()或computerJudge()這兩個判斷是否要牌的函數
            isWant = judge(myPoint, bPoint) 
            if (isWant == True):
                #如果電腦或玩家要牌，讓使用者輸入並更新當前點數
                myPoint = myPoint + transferPoint(input())
                #判斷點數是否超過 10.5
                over, myPoint = justPoint(myPoint)
            return isWant, myPoint, over


副程式-transferPoint():將牌的點數轉換 

        def transferPoint(n):
            if n == 'A':
                return 1
            elif n == 'J' or n == 'Q' or n == 'K':
                return 0.5
            else:
                return float(n)
副程式-justPoint():判斷點數是否超過 10.5 點 

        def justPoint(point):
            #初始化 over 為 False
            over = False
            #如果點數超過 10.5  
            if (point > 10.5): 
                #點數歸0 
                point = 0 
                #over設為True，表示超過了 
                over = True 
            #回傳是否超過和當前的點數  
            return over, point
副程式-playerJudge():玩家是否要牌 

        def playerJudge(a, b):
            #讓玩家輸入是否要牌，'Y' 代表要牌
            isWant = input() 
            if (isWant == 'Y'):
                return True 
            else:

副程式-computerJudge():判斷電腦是否要牌 

        def computerJudge(a, b):
            #如果電腦的點數小於玩家的點數，電腦要牌
            if (a < b):  
                return True 
            #否則如果電腦的點數小於或等於8，電腦也要牌
            elif (a <= 8):  
                return True 
            #否則其他情況下，電腦不要牌 
            else: 
                return False
