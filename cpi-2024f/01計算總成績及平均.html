01 計算總成績及平均


使用者輸入名字(string)、學號(int)、以及國文Chinese、計算機概論CS、計算機程式設計PD三個科目的成績(int)。

請輸出學生名字、學號，並計算其平均成績與總分。

1. 輸入學生名字
2. 輸入學生學號
3. 輸入學生國文成績
4. 輸入學生計算機概論成績
5. 輸入學生計算機程式設計成績

2. ID:學生學號
3. Average:學生三科成績之平均 (保留整數即可)

資料型態

Python 的內建型態主要分為以下三種:

 1. 數值型態：int、float、bool
 2. 字串型態：str、chr
 3. 容器型態：list、dict、tuple

數值型態
int 整數型態 (EX:1、2、3、4、5)
float 浮點數型態 (EX:1.12、0.01、3.45)
bool 布林值 (True、False)
字串型態
str 字串:用單引號 ' 或雙引號 " 包起來的文字資料。
EX: 'ABCDE' 、 "ABCDE"
chr 字元:組成字串的元素，用單引號 ' 包起字元。
EX: 'A'、'B'、'C'、'D'
容器型態
list 列表:用來儲存一組有序的元素。列表是可變的，可以在列表中新增、刪除和修改元素。
用 [ ] 表示
EX: [1,2,3,4]
dict 字典:每一個元素都由鍵 (key) 和值 (value) 構成，結構為key: value。
用 { key:value } 表示
EX: {1:'A',2:'B',3:'C',4:'D'}
tuple 元組(數組):跟list一樣用來儲存一組有序的元素，但是元組是不可變的。
用 ( ) 表示
EX: (1,2,3,4,5)

解答

主程式 

              def main():
                #輸入學生姓名，用input()都是字串型態，存進name變數
                name = input()
                #輸入學生學號，並轉換為整數資料型態，int()就是轉換的語法，存進id變數
                id = int(input())
                #輸入國文成績，並轉換為整數資料型態，存進chinese變數
                chinese = int(input()) 
                #輸入計算機概論成績，並轉換為整數資料型態，存進com變數
                com = int(input())
                #輸入計算機程式設計成績，並轉換為整數資料型態，存進pd變數
                pd = int(input())
                #將三科分數加起來，存進total變數
                total = chinese + com + pd
                #計算平均分數，存進average變數
                average = total / 3 

                # "%s"% 代表將name變數格式化為字串印出
                print("Name:%s"%name) 
                # "%d"% 代表將ID變數格式化為十進制整數印出
                print("ID:%d"%id) 
                # "%.0f" 代表將average變數格式化為float，且四捨五入取到整數印出
                print("Average:%.0f"%average)
                # 將total變數以十進制整數方式印出
                print("Total:%d"%total)
