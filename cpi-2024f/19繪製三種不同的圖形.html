19 繪製三種不同的圖形


N代表圖形的行數，而N (3 <= N <= 9)

第一種圖形為直角三角形：
共有 N 行
第一行輸出1
第二行輸出121
第三行輸出12321
以此類推，第 i 行輸出 1… i-1 + i + i-1…1 (i 為 1~N中的任一行)
每一行共有2 * i - 1個數字
例如，當N = 5時，輸出圖形為:
1
121
12321
1234321
123454321
第二種圖形為正三角形：
共有N行
第一行輸出 (N-1)個 _ + 1 + (N-1)個 _
第二行輸出 (N-2)個 _ + 121 + (N-2)個 _
第三行輸出 (N-3)個 _ + 12321 + (N-3)個 _
以此類推，第 i 行輸出 (N-i)個 _ + (1...i-1 + i + i-1…1) + (N-i)個 _ (i 為 1~N中的任一行)
每一行共有2 * N - 1個字元，包含(N-i)*2個_ 與 2 * i - 1個數字
例如，當N = 4時，輸出圖形為:
___1___
__121__
_12321_
1234321
第三種圖形為倒三角形：
共有N行
第一行輸出 0個 _ + 1...N-1 + N + N-1…1 + 0個 _
第二行輸出 1個 _ + 1...N-2 + N-1 + N-2…1 + 1個_
第三行輸出 2個 _ + 1...N-3 + N-2 + N-3…1 + 2個 _
以此類推，第 i 行輸出 (i-1)個 _ + (1...N-i+1) + (N-i…1) + (i-1)個 _ (i 為 1~N中的任一行)
每一行共有2 * N - 1個字元，包含(i-1)*2個_ 與 (2*(N-i)+1) 個數字
例如，當N = 4，輸出圖形為:
1234321
_12321_
__121__
___1___

輸入說明 輸出說明
1. 輸入整數M，1<=M<=3，代表接下來要畫的圖形種類。
 M=1，為第一種圖形 直角三角形 (參考 範例輸出說明 1)
 M=2，為第二種圖形 正三角形 (參考 範例輸出說明 2)
 M=3，為第三種圖形 倒三角形 (參考 範例輸出說明 3)
2. 輸入整數 N ，代表這個圖形有N行，N (3 <= N <= 9) 		根據輸入，畫出對應的圖形。

副程式-equilateralTriangle():正三角形 

              def equilateralTriangle(n):
                  # row的範圍 1~n
                  for row in range(1, n+1): 
                      printBottomLine(row, n)
                      printPalindrome(row)
                      printBottomLine(row, n)
                      print()
副程式-invertedTriangle():倒三角形 

            def invertedTriangle(n):
                #row的範圍 n~1 (會遞減)
                #當row = n時，就不會畫出底線
                for row in range(n, 0, -1): 
                    printBottomLine(row, n)
                    printPalindrome(row)
                    printBottomLine(row, n)        
                    print()

副程式-printPalindrome():畫出數字回文 
 
          def printPalindrome(row):
              for i in range(1, row + 1):
                  #印出數字1~row
                  print(i, end="")
              for i in range(row - 1, 0, -1):
                  #印出數字(row-1) ~ 1
                  print(i, end="")
副程式-printBottomLine():畫出底線 
 
          def printBottomLine(row, n):
              for i in range(0, n - row):
                  #會印出(n-row)個 _
                  print('_', end="")
