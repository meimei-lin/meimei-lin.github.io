03 數值運算


使用者請分別輸入兩個數 num1、num2，並求出兩數的和(Sum)、差(Difference)、乘積(Product)、商(Quotient)。

結果須輸出到小數點第二位。


輸入說明 輸出說明
1. 輸入數字num1
2. 輸入數字num2 		1. Sum:num1+num2
2. Difference:num1-num2
3. Product:num1*num2
4. Quotient:num1/num2

Input/Output


Sample Input Output
25
2
 Sum:27.00
Difference:23.00
Product:50.00
Quotient:12.50
-1
6
Sum:5.00
Difference:-7.00
Product:-6.00
Quotient:-0.17
0
9
 Sum:9.00
Difference:-9.00
Product:0.00
Quotient:0.00

Hints









                            
def main():

                                #輸入第一個數

                                #輸入第二個數


                                #輸出經 +、-、*、/ 計算後的結果

解答

主程式

                    
def main():

                        #輸入第一個數字，並轉換為整數型態，存進num1變數中

                        num1 = int(input())

                        #輸入第二個數字，並轉換為整數型態，存進num2變數中

                        num2 = int(input())


                        #印出兩數相加的結果，"%.2f"%代表將num1+num2的結果取到小數點後第二位印出

                        #若答案是整數，小數點後兩位會印出0 (EX: xx.00)

                        print("Sum:%.2f"%(num1+num2))

                        #印出兩數相減的結果

                        print("Difference:%.2f"%(num1-num2))

                        #印出相乘的結果

                        print("Product:%.2f"%(num1*num2))

                        #印出相除的結果

                        print("Quotient:%.2f"%(num1/num2))
呼叫主程式

                    
main()
