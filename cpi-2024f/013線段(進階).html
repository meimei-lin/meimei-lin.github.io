013 線段（進階）

請計算出(L1、L2、L3…Ln)N條線在 X 軸上所涵蓋的長度(不含重複線段)。
例如: L1(x1,x2)表示L1線段為 X 軸上點 x1 到點 x2 的線。
依序輸入起點和終點(起點 < 終點)，且較小的起點先輸入。


輸入說明 輸出說明
1. 輸入總共有多少條線段
（整數N, 1 <= N <= 40）
2. 輸入 L1 的 x1, x2 座標
（整數M, -20 <= M <= 20）
3~N. 重複第二行步驟直到Ln也輸入完成
 1. 輸出N條線段所涵蓋的長度（整數）

Input/Output


第一行：總共有3條線段
第二行：L1 的 (x1,x2) 座標為 (-1,1) ->（最小的起點先輸入，並以空格隔開）
第三行：L2 的 (x1,x2) 座標為 (0,2) ->（最小的起點先輸入，並以空格隔開）
第四行：L3 的 (x1,x2) 座標為 (1,3) ->（最小的起點先輸入，並以空格隔開）

for 迴圈



目的：讓指定的某段敘述在條件符合的情況下一直重複執行

語法： 

            for x in sequence:
             #要執行的敘述

for 和 in 是Python的關鍵字，這兩個關鍵字之間會放我們自訂的變數，sequence是放一組序列資料， 迴圈會依序從序列（例如: 一連串清單資料，一組數字或一個名單）裡取得元素，並將元素指派給前面自訂的變數，並執行迴圈裡的內容。

            for i in Abook_sequence:
              print(i)

設定初始值、終止值，以及每次執行迴圈後控制變數遞增或是遞減的變化。
for x in range(起始值, 終止值, 遞增(減)值)
range(10)：產生從0到9的整數序列。
range(1, 11)：產生從1到10的整數序列(未指定遞增值的情況下，其遞增值預設為1)。
range(0, 10, 2)：產生從0, 2, 4, 6, 8的整數序列(遞增值為2)。

解答

主程式 

                def main():
                    #輸入總共有多少條線段
                    n = int(input())
                    #定義變數lines，型態為list
                    lines = []
                    for i in range(n):
                        #輸入Ln的座標，用空格分割字串
                        line_point = input().split(' ')
                        #把座標範圍 -20~20 變成 0~40，存進變數x1和x2
                        x1, x2 = int(line_point[0])+20, int(line_point[1])+20
                        #新增座標到變數lines
                        lines.append([x1, x2])

                    linear(lines)
