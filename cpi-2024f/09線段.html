09 線段

依序輸入a,b,c三條線的座標，計算出 a,b,c 三條線在 X 軸上所涵蓋的長度(不含重複線段)。
例如: a(x1,x2)表示 a 線段為 X 軸上點 x1 到點 x2 的線， 依序輸入起點和終點
(起點 < 終點)，且較小的起點先輸入。


輸入說明 輸出說明
1. 輸入 a 的 x1 座標 (整數, -20 ~ 20)
2. 輸入 a 的 x2 座標 (整數, -20 ~ 20)
3. 輸入 b 的 x1 座標 (整數, -20 ~ 20)
4. 輸入 b 的 x2 座標 (整數, -20 ~ 20)
5. 輸入 c 的 x1 座標 (整數, -20 ~ 20)
6. 輸入 c 的 x2 座標 (整數, -20 ~ 20)		 1. 輸出三條線段所涵蓋的長度 (整數)

Input/Output


第一行：a 的 x1 座標為 -1 -> A Line起點 : -1 (最小的起點，先輸入)
第二行：a 的 x2 座標為 1
第三行：b 的 x1 座標為 0 -> B Line起點 : 0 (次之)
第四行：b 的 x2 座標為 2
第五行：c 的 x1 座標為 1 -> C Line起點 : 1 (最後輸入)
第六行：c 的 x2 座標為 3

              def linear(a1,a2,b1,b2,c1,c2):
                #定義變數s為a線段的起點
                s = a1 
                #定義變數e為a線段的終點
                e = a2
                #定義變數ans，初始化為0，用來存總長度
                ans = 0 

                #這裡的if-elif一定會更新到變數e
                #如果a和b線段沒有重疊
                if b1 >= e: 
                    #算出a線段的長度並累加到 ans
                    #獨立計算沒有重疊的長度
                    ans += (e-s) #相當於ans = ans + (e-s)
                    #更新起點與終點為b線段的座標
                    s = b1
                    e = b2
                #否則如果，a和b線段有重疊
                elif b2 > e:
                    #更新終點為b線段的x2座標
                    #此時s還是a線段的x1座標:a1
                    e = b2

                #如果c和更新後的終點e沒有重疊
                if c1 >= e:  
                    ans += (e-s) #ans = ans + (e-s)
                    #更新起點與終點為c線段的座標
                    s = c1
                    e = c2
                #否則如果，c線段和更新後的終點e有重疊
                elif c2 > e:
                    #更新終點為c線段的x2座標
                    e = c2
            
                #計算總長度
                ans += (e - s)
                print(ans) #印出
