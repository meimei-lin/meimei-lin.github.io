05 判斷何種三角形



當三個邊長能構成三角形時，判斷該三角形是否為正三角形，若否，則判斷是否為等腰三角形。若皆非正三角形或等腰三角形，再判斷該三角形為鈍角三角形、銳角三角形或直角三角形。

能否構成三角形與三角形種類的條件說明：
 1. 不能成為三角形：任兩邊和不大於第三邊，或任一邊長不大於0。
 2. 正三角形：三個邊相等。
 3. 等腰三角形：任兩邊相等，且平方和大於第三邊的平方。
 4. 鈍角三角形： 最長邊平方大於兩短邊平方和。
 5. 銳角三角形： 最長邊平方小於兩短邊平方和。
 6. 直角三角形： 最長邊平方等於兩短邊平方和。

注意： 這題必須寫一個運算的function：def getTriangle(a, b, c)

1. 輸入第一邊邊長
2. 輸入第二邊邊長
3. 輸入第三邊邊長
(輸入的三個邊長都是int型態的) 		若結果為：
1. 不能成為三角形：輸出 not a triangle
2. 正三角形：輸出 equilateral triangle
3. 等腰三角形：輸出 isosceles triangle
4. 鈍角三角形：輸出 obtuse triangle
5. 銳角三角形：輸出 acute triangle
6. 直角三角形：輸出 right triangle

                def getTriangle(a, b, c): #有三個接受參數，代表輸入的三個邊

                    #如果a、b、c三個邊其中一個邊小於0
                    if a < 0 or b < 0 or c < 0:
                        print("not a triangle") #印出 not a triangle

                    #否則如果，任兩個邊的和小於等於(不大於)第三邊                     
                    elif a + b <= c or a + c <= b or b + c <= a:
                        print("not a triangle") #印出 not a triangle

                    #否則如果，a等於b 且 b等於c，代表三邊等長，為正三角形
                    elif a == b and b == c:
                        print("equilateral triangle") #印出 equilateral triangle

                    #否則如果，任兩個邊相等 且 (相等兩個邊平方和) 大於 第三邊的平方
                    #為等腰三角形
                    elif ((a == b and ((a*a + b*b) > c*c)) or (b == c and ((b*b + c*c) > a*a)) or (a == c and (a*a + c*c) > b*b)):
                        print("isosceles triangle") #印出 isosceles triangle

                    #否則，都不滿足以上三種情況的話，要判斷是鈍角、銳角、直角三角形
                    else:

                        #如果a是最長的邊，此時兩短邊是b、c
                        if a > b and a > c: 

                            #如果兩短邊b、c的平方和 小於 最長邊a的平方
                            #為鈍角三角型
                            if (b*b + c*c) < (a*a): 
                                print("obtuse triangle") #印出 obtuse triangle

                            #否則如果，兩短邊b、c的平方和 大於 最長邊a的平方
                            #為銳角三角形
                            elif (b*b + c*c) > (a*a):
                                print("acute triangle") #印出 acute triangle

                            #否則如果，兩短邊b、c的平方和 等於 最長邊a的平方
                            #為直角三角形
                            elif (b*b + c*c) == (a*a):
                                print("right triangle") # 印出 right triangle

                        #否則如果，b是最長的邊，此時兩短邊是a、c
                        elif b > a and b > c: 
                            if (a*a + c*c) < (b*b):
                                print("obtuse triangle")
                            elif (a*a + c*c) > (b*b):
                                print("acute triangle")
                            elif (a*a + c*c) == (b*b):
                                print("right triangle")

                        #否則如果，c是最長的邊，此時兩短邊是a、b
                        elif c > a and c > b: 
                            if (a*a + b*b) < (c*c):
                                print("obtuse triangle")
                            elif (a*a + b*b) > (c*c):
                                print("acute triangle")
                            elif (a*a + b*b) == (c*c):
                                print("right triangle")
