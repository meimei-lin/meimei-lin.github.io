011 買書

有A、B、C三種書本，一顧客欲購買A:ｘ本、B:ｙ本、C:ｚ本（ｘ、ｙ、ｚ為使用者輸入），請計算A、B、C各花費多少， 以及總共需花費多少錢？

每本書的費用，最後以無條件進位法到整數。

A、B、C三本書價格如下：
定價 1~10本 11~20本 21~30本 31本以上
A 380 原價 打A1折 打A2折 打A3折
B 1200 原價 打B1折 打B2折 打B3折
C 180 原價 打C1折 打C2折 打C3折


輸入說明 輸出說明
1. A 書本數量(整數,0~100),A1,A2,A3
2. B 書本數量(整數,0~100),B1,B2,B3
3. C 書本數量(整數,0~100),C1,C2,C3
(A1~C3為整數0~100，若85為85折，即為*0.85)
 1. 花費最多的書本名稱、費用(中間以逗號隔開)
2. 花費中間的書本名稱、費用(中間以逗號隔開)
3. 花費最少的書本名稱、費用(中間以逗號隔開)
4. 總費用
(如果兩本書費用相同，按照A>B>C的順序輸出)
(總費用為每本書最後整數費用的加總)

Input/Output


第一行：A書本的數量:6, A1:95, A2:80, A3:75
第二行：B書本的數量:12, B1:95, B2:80, B3:70
第三行：C書本的數量:30, C1:90, C2:80, C3:75
A: 6*380=2280(10本以下，沒有打折)
B: 12*1200*0.95=13680(11~20本，打B1折)
C: 30*180*0.8=4320(21~30，打C2折)
費用由高到低輸出
總費用：13680+4320+2280=20280

分割字串（split）

在Python中，可以使用split()函數來將一個字串分割成多個子字串，分割後的子字串會轉換成list列表型態，裡面的元素是字串型態的。

語法：str.split(' ') # ' ' 填入要分割的符號

            str1 = "A,12,99,89,79"
            str1_split = str1.split(',')
            print(str1_split)

輸出：['A', '12', '99', '89', '79']

數學函數（math）


Python的math函數提供了許多常用的數學函式，例如三角函數、四捨五入、指數、對數、平方根、總和...等，都能夠透過 math 函數來進行運算。

需要import math才能使用

較常用：
ceil(x): 無條件進位到整數
floor(x): 無條件捨去到整數
fabs(x): 回傳 x 的絕對值（浮點數）
gcd(x, y): 回傳 x 和 y 的最大公因數
pow(x, y): 回傳 x 的 y 次方
sqrt(x): 回傳 x 的平方根

副程式-bookA_price() 

              def bookA_price(bookA,A1,A2,A3):
                #定義priceA變數，初始化為0，代表計算後的書本A價錢
                priceA = 0 
                #如果數量10本(含)以下
                if(0 <= bookA <= 10): 
                    priceA = 380*bookA #以原價計算
                #否則如果，11~20本，打A1折
                elif(11 <= bookA <= 20): 
                    priceA = (380*bookA)*(A1/100)
                #否則如果，21~30本，打A2折
                elif(21 <= bookA <= 30):
                    priceA = (380*bookA)*(A2/100)
                #否則如果，31本以上，打A3折
                #上限100本
                elif(30 < bookA <= 100):
                    priceA = (380*bookA)*(A3/100)
                #回傳priceA的結果
                return priceA
